## 🎨 Stack

- HTML5 + CSS3
- Vanilla JavaScript
- Google Fonts (Fraunces, DM Sans, JetBrains Mono)
- Sistema de imágenes automático

---

## 📦 Features

- ✅ Portfolio multi-página con navegación fluida
- ✅ Dark/Light mode
- ✅ Cursor custom
- ✅ Reveal animations
- ✅ Filtros de proyectos
- ✅ Responsive design
- ✅ SEO optimizado
- ✅ Sistema automático de carga de imágenes
